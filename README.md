# What is it?

A custom repository for custom assets for flipper zero and another devices (WIP: pls wait...)

# Current support

* Momentum Custom Firmware (MCF)
* Flipper Official Firmware (FOF)

# Copy files

* For MCF you need to navigate to:
    * `SD Card/asset_packs` and copy/paste your chosen `directory` from `./momemtum` into that
* For the FOF:
    * `SD Card/dolphin` and copy/paste `manifest.txt` if you want to copy all directories from `./o_firmware`
